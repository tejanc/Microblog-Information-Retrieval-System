Microblog information retrieval system
======================================

This project is an implementation of an Information Retrieval (IR) system based on a 
collection of documents (Twitter messages). The collection is a subset of the data 
used in the TREC 2011 Microblog retrieval task. Each line contains the id and the 
Twitter message and the actual message.

It is very simple and easy to use

Features
--------

Techniques used in Preprocessing
Techniques used in indexing
Technques used in ranking and retrieval
Evauluating the script
